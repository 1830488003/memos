新手使用教程
📖 点我展开 MemOS 使用教程（推荐新手先看）

1️⃣ 先做什么？

先去 MemOS 官网获取 API Key，没有 API Key 插件是无法正常工作的。

2️⃣ API Key 在哪拿？

① 打开 MemOS API Key 页面

② 登录账号

③ 点击“快速接入 Cloud API”

④ 复制你的 API Key（通常是 mpg- 开头）

3️⃣ 在插件里怎么填？

① 把 API Key 粘贴到下面的“API 密钥”输入框

② 点击“保存配置”

③ 再点击“测试连接”，确认显示连接成功

4️⃣ 下面几个开关怎么理解？

- 保存记忆：是否把你后续对话写进 MemOS

- 自动检索记忆：是否在聊天时自动去查记忆

- 注入记忆类型：控制把哪些类型的记忆塞进 prompt

- 知识库配置：适合只想查资料、不想存聊天记忆的用户

💡 推荐给新手的最简单用法： 填 API Key → 保存配置 → 测试连接 → 保持默认勾选直接使用。

📚 知识库配置
📖 知识库使用教程（点击展开）

1️⃣ 知识库的作用

知识库可以存储大量文档资料（PDF、Word、TXT等），AI在回答问题时可以结合知识库内容，让回答更准确、更专业。

2️⃣ 如何添加知识库资料

① 打开知识库管理页面：点击前往 →

② 创建知识库（免费用户可创建10个）

③ 上传你的文档资料（PDF、DOCX、TXT等）

④ 复制知识库的ID，填到下方框中

3️⃣ 如何在插件中使用

① 复制知识库ID（以"id_"开头）

② 粘贴到下方"知识库ID"输入框

③ 填写名称方便自己识别（如"产品手册"）

④ 勾选启用，保存配置即可生效
