📖 使用说明

▼

配置步骤：

  1. 在 MemOS Cloud 控制台获取 API Key
  2. 填写上述配置并保存
  3. 点击"测试连接"验证配置

功能说明：

  • 自动保存：对话中的消息会自动保存到 MemOS（用户和AI消息都会保存）
  • 自动检索：发送用户消息时自动检索相关记忆并注入到回复生成中
  • 手动保存：手动输入内容保存到 MemOS
  • 相关度阈值：只显示相关度高于此值的记忆

数据统计：

  • 记忆总数：从 API 获取的当前记忆总数
  • 已保存消息：本地已处理的对话消息数量
  • 检索次数：手动搜索和自动检索的次数

API 端点：

  • 搜索: POST /search/memory
  • 添加: POST /add/message
  • 获取: POST /get/message
